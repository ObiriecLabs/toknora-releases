# BURNRATE

**Claude Code Analytics** — Track costs, sessions, and performance of your Claude Code usage.

## Download

👉 **[Latest Release](https://github.com/ObiriecLabs/burnrate-releases/releases/latest)**

| Platform | File |
|---|---|
| macOS | `BURNRATE-{version}-mac.dmg` |
| macOS (auto-installer) | `BURNRATE-installer-mac.zip` |
| Windows | `BURNRATE-{version}-win-setup.exe` |

## About

BURNRATE reads your Claude Code session files (`~/.claude/projects/`) and shows you
interactive analytics: cost breakdown, token usage, session activity, optimization tips, and more.
